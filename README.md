# Color Blue Store

Página web de uma loja fictícia de produtos, desenvolvida como exercício de front-end.

## Tecnologias

- HTML5
- CSS3
- Bootstrap 5

## Funcionalidades

- Página inicial com banner e vitrine de produtos
- Seção de equipe e workshops
- Layout responsivo

## Como executar

Abra o arquivo `index.html` diretamente no navegador.
